### Shortcuts

Key          | Action        |
:----------- | :-----------: |
CTRL + F     | Fit Image     |
CTRL + +     | Zoom +        |
CTRL + -     | Zoom -        |
CTRL + o     | Open          |
CTRL + 0     | Original Size |
SHIFT + F1   | Help          |
