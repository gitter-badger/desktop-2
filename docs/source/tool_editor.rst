SiDE Editor
===================

.. image:: img/side-edit.png

Functions
---------

- Indicate file changing
- Line highlighting / numbering
- show matching brackets
- save window position and size
- warning if closing without save

Shortcuts
---------

===========    =====  ==============
    Keys               Function
--------------------  --------------
 Mod    Key
===========    =====  ==============
CTRL            S      Save
CTRL SHIFT      S      Save as
CTRL            N      New
CTRL            ,      Settings
CTRL            O      Open File
CTRL            Z      Undo
CTRL            Y      Redo
SHIFT           F1     Open help
===========    =====  ==============
